Earn money with the Sidebar for Amazon

To prepare to earn money with the Sidebar for Amazon, you need to do the following:

A:
  1. Register for an Amazon Associate account.
  2. Get the Amazon Associate Tag from your Amazon Associate account(s).
  3. Input your Associate Tag(s) into the Sidebar for Amazon module (see screenshot A 3).
B:
  1. Register to the Amazon Product Advertising API.
  2. Get the Access Key ID and Secret Access Key from your Amazon Product Advertising API account.
  3. Input your Access Key ID and Secret Access Key into the Sidebar for Amazon module (see screenshot B 3).

If you already have an Amazon account for making purchases, you can use this buyer account to register as a seller with Amazon Associates and the Amazon Product Advertising API.

A: Getting Your Associate Tags

To become an Amazon Associate, go to the Amazon Associates main page for your regional locale. One primary requirement for becoming an Amazon Associate is that you furnish the URL web address of your site during registration. If your site is not yet public but you want to test Sidebar for Amazon against the API, you still need to supply a URL during the registration process.

Canada: https://associates.amazon.ca
China: https://associates.amazon.cn
France: https://partenaires.amazon.fr
Germany: http://partnernet.amazon.de
Italy: https://programma-affiliazione.amazon.it
Japan: https://affiliate.amazon.co.jp
Spain: https://afiliados.amazon.es
United Kingdom: https://affiliate-program.amazon.co.uk
United States: http://affiliate-program.amazon.com

From the Amazon Associates account, copy the Associate Tag to the Sidebar for Amazon module corresponding field (screenshot A 3). You may find the Associate Tag in your email Inbox, too, because the Associate Tag is usually sent to you by email when you first register as an Associate.

Important: Your Associate Tag will only earn you referral fees from sales in the same locale as your Associate Tag. For example, to earn referral fees from Amazon.co.uk, you must be a member of the UK Associates program and use that UK Associate Tag in the Sidebar for Amazon module. You must sign up for the Amazon Associates program in each Amazon locale for which you will use the Sidebar for Amazon module!

B: Getting Your Access Key and Secret Access Key

Amazon's Product Advertising API is a free service that provides you with access to Amazon's extensive product catalog data. It is very important that you register to Amazon’s Product Advertising API using the same email address used by your main Amazon Associate account!

1. Login to your main Amazon Associates account and click on the tab "Product Advertising API" (recommended);

or go to the Amazon Web Service main page for your locale:

Canada: https://associates.amazon.ca/gp/flex/advertising/api/sign-in.html
China: https://associates.amazon.cn/gp/flex/advertising/api/sign-in.html
France: https://partenaires.amazon.fr/gp/flex/advertising/api/sign-in.html
Germany: https://partnernet.amazon.de/gp/flex/advertising/api/sign-in.html
Italy: https://programma-affiliazione.amazon.it/gp/flex/advertising/api/sign-in.html
Japan: https://affiliate-program.amazon.com/gp/flex/advertising/api/sign-in-jp.html
Spain: https://afiliados.amazon.es/gp/flex/advertising/api/sign-in.html
United Kingdom: https://affiliate-program.amazon.co.uk/gp/flex/advertising/api/sign-in.html
United States: https://affiliate-program.amazon.com/gp/flex/advertising/api/sign-in.html

2. Click Sign Up to register for a Product Advertising API account.

3. From the Welcome page, click Manage Your Account.

4. Under Access Identifiers, choose Click Here.

5. Copy the Access Key ID and Secret Access Key to the corresponding fields in the Sidebar for Amazon module (screenshot B 3).
